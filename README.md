# AI Interview Tool

An interactive AI-powered interview practice application with real-time feedback and speech recognition.

## Quick Start

```bash
# Clone or download the project
# npx degit rishavtiwari22/temp-project my-app
# cd my-app

# Install dependencies
npm install

# Configure API Key (IMPORTANT!)
# 1. Copy .env.example to .env
# 2. Get your FREE API key from: https://openrouter.ai/keys
# 3. Add your key to .env file:
#    VITE_OPENROUTER_API_KEY=your_key_here

# Start development server
npm run dev
```

## Configuration

The app uses OpenRouter API (free tier) for AI features. Without an API key, the app will still work but with limited AI features (generic responses instead of contextual AI feedback).

To get full AI features:

1. Visit https://openrouter.ai/keys
2. Sign up for a free account
3. Generate an API key
4. Add it to your `.env` file

## Notes

- The app works offline for basic features
- AI features require internet connection and valid API key
- Speech recognition works in Chrome/Edge browsers
